### Manual installation

```bash
apt update && apt install git libcairo2 -y
git clone https://github.com/Splaueef/hikka
cd Hikka
pip install -r requirements.txt
python3 -m hikka
````

If you are on **VPS / VDS**, add:

```
--proxy-pass
```

to open SSH tunnel to the web interface.

Or use:

```
--no-web
```

to complete setup in console.
